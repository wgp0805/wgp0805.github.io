docker安装及使用-window环境

一、环境准备

  1. windows 10以上
  2. CPU支持虚拟化技术

二、安装docker

1、访问docker官网下载docker安装器

点击下载即可

2、安装前准备

因为docker desktop 是可以部署在windows 上来运行docker的应用服务的，其基于windows 的 Hyper-V服务和WSL2内核的Windows上创建一个子系统（linux）,从而实现其在windows上运行docker。所以电脑上需要开启Hyper-V服务和安装WSL2。

在系统中搜索控制面板，打开程序，点击启用或关闭windows功能，开启Hyper-V,windows虚拟机监控程序平台，适用于Liunx的windows的子系统

配置完毕后，重启电脑

备注：

由于 Windows 家庭版没有 Hyper-V，因此必须先安装它。
